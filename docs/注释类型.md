# 常见编程语言注释类型汇总 (Comment Styles Reference)

本文档整理了 ALS (Auto-Language-Switch) 插件计划支持或已经支持的主流编程语言及其注释规范。

| 语言家族 | 典型语言 | 常见后缀 | 单行注释 | 多行 (块) 注释 | 文档/特殊注释 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **C-Style** | C, C++, C#, Java, JS, TS, Rust, Go, Swift, Kotlin, Scala, Dart | `.c`, `.cpp`, `.cs`, `.java`, `.js`, `.ts`, `.rs`, `.go`, `.swift`, `.kt`, `.scala`, `.dart` | `//` | `/* ... */` | Java/JS/TS: `/** ... */`<br>Rust: `///` (Doc), `//!` (Module)<br>C#: `///` (XML Doc) |
| **Script (Hash)** | Python | `.py` | `#` | 无原生块注释<br>(通常用 `"""..."""` 或 `'''...'''` 字符串代替) | Docstring: `"""..."""` |
| **Script (Hash)** | Shell (Bash), YAML, Dockerfile, Makefile, Perl, Ruby, R | `.sh`, `.yaml`, `.yml`, `Dockerfile`, `Makefile`, `.pl`, `.rb`, `.r` | `#` | Ruby: `=begin ... =end`<br>Perl: `=begin ... =cut` | Ruby: RDoc |
| **Web** | HTML, XML, SVG | `.html`, `.xml`, `.svg` | 无 | `<!-- ... -->` | - |
| **Web** | CSS, LESS, SCSS | `.css`, `.less`, `.scss` | CSS: 无 (标准只支持 `/* */`)<br>LESS/SCSS: `//` | `/* ... */` | - |
| **SQL** | SQL (Standard), PL/SQL, T-SQL | `.sql` | `--` | `/* ... */` | - |
| **Lua** | Lua | `.lua` | `--` | `--[[ ... ]]` | - |
| **Haskell** | Haskell | `.hs` | `--` | `{- ... -}` | - |
| **Matlab** | Matlab, Octave | `.m` | `%` | `%{ ... %}` | - |
| **LaTeX** | LaTeX, TeX | `.tex` | `%` | 无 (通常用包支持) | - |
| **Lisp** | Clojure, Lisp, Scheme | `.clj`, `.lisp`, `.scm` | `;` | `#| ... |#` | - |
| **Batch** | Windows Batch | `.bat`, `.cmd` | `REM` 或 `::` | 无 | - |
| **PowerShell**| PowerShell | `.ps1` | `#` | `<# ... #>` | - |
| **VB** | Visual Basic, VBA, VBScript | `.vb`, `.vbs` | `'` (单引号) | 无 | - |
| **Erlang** | Erlang | `.erl` | `%` | - | - |
| **Fortran** | Fortran | `.f90` | `!` | - | - |

## ALS 插件支持策略

### 1. 通用匹配 (Heuristic)
大部分现代语言深受 C 语言影响，因此默认策略应优先覆盖：
*   **Line**: `//`
*   **Block**: `/* ... */`

### 2. 特殊处理 (Special Handling)
需要针对特定 `languageId` 启用额外的匹配规则：

#### Python
*   **Line**: `#`
*   **Block**: `"""` 和 `'''` (作为 Docstring，视为中文注释区域)

#### HTML/XML/Markdown
*   **Block**: `<!-- ... -->`

#### Shell/YAML/Config
*   **Line**: `#`

#### Lua/SQL
*   **Line**: `--`
*   **Block**: `--[[ ... ]]` (Lua)

### 3. VS Code Language IDs 映射表
在编写插件代码时，应使用 VS Code 标准的 `languageId`：

```typescript
const CONFIG = {
    // C-Style
    'c': { line: '//', block: ['/*', '*/'] },
    'cpp': { line: '//', block: ['/*', '*/'] },
    'java': { line: '//', block: ['/*', '*/'] },
    'javascript': { line: '//', block: ['/*', '*/'] },
    // ...

    // Python
    'python': { line: '#', block: ['"""', '"""'] }, // 需特殊处理 '''

    // Shell
    'shellscript': { line: '#', block: null },
    
    // HTML
    'html': { line: null, block: ['<!--', '-->'] },
    
    // LaTeX
    'latex': { line: '%', block: null }
};
```
